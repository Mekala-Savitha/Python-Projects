Quiz Application

Description: 
The Quiz Application is a Python project that allows users to answer multiple-choice questions and receive a score at the end. 
It helps beginners practice Python concepts such as conditions, and user input handling.

Features: 
• Displays multiple-choice questions. • Accepts user answers.
• Checks whether the answer is correct or incorrect.
• Calculates the total score. 
• Shows the final result after all questions are completed. 
• Easy to customize by adding new questions.

Concepts Used:
• Variables 
• User Input
• Conditional Statements (if-else)
• Basic Output (print())

How It Works:
1.The program displays a question with multiple options.
2.The user selects an answer.
3.The program checks the answer.
4.The score is updated if the answer is correct.
5.After all questions are completed, the final score is displayed.

Learning Outcome:
This project helps beginners understand Python fundamentals and improve problem-solving skills by building an interactive command-line application.
